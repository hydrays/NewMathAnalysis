## Problem

用微积分基本定理（牛顿-莱布尼茨公式）计算 $\displaystyle\int_0^1 x^2\,dx$。

## Solution 1

### Hint

求 $f(x) = x^2$ 的一个原函数，然后代入上下限作差。

### Answer

$f(x) = x^2$ 的一个原函数为 $F(x) = \dfrac{x^3}{3}$。

由微积分基本定理：

$$
\int_0^1 x^2\,dx = F(1) - F(0) = \frac{1}{3} - 0 = \frac{1}{3}
$$
