## Problem

判断数列 $a_n = n = \{1, 2, 3, 4, \cdots\}$ 是否收敛，并说明理由。

## Solution 1

### Hint

观察数列各项是否有界，无界数列必然发散。

### Answer

数列 $a_n = n$ 不收敛（发散至正无穷）。

数列不断增长，无法趋近于任何有限的固定值，因此"静"不下来。

严格地说，若极限为 $A$，则对 $\varepsilon = 1$，存在 $N$ 使得当 $n > N$ 时 $|n - A| < 1$，即 $A - 1 < n < A + 1$。但 $n$ 可以任意大，当 $n > A + 1$ 时不等式不成立，矛盾。故数列发散。

更简洁地，有界数列才可能收敛，而 $\{n\}$ 无界，因此发散。
